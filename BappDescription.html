This Burp Suite extension, JWT Monitor, keeps track of a given JWT tokens expiration and substitutes any existing JWT token in outgoing requests with the given one. If autopilot mode is enabled, the extension automatically halts task execution if the JWT token is set to expire in the next minute. Task execution will resume automatically when a new JWT token with at least three minutes of validity remaining is provided.

Features

  • Monitors a provided JWT token for expiration.
  • Replaces any existing JWT token in outgoing requests with the monitored one.
  • Autopilot mode which pauses task execution when the token is about to expire.
  • Task execution automatically resumes when a new JWT token with sufficient validity is supplied.

Usage

Once the extension is loaded, supply it with a JWT token. The extension will then monitor this token for its expiry. If another token is included in outgoing requests, it will replace it with the provided one.

To use the autopilot feature, activate it in the extension settings. If the monitored token will expire within the next minute, task execution will pause. Task execution will resume automatically when a new JWT token with at least three minutes of validity is supplied.
